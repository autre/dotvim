
Installation:
===

    git clone git://github.com/autre/dotvim.git ~/.vim # clone this repo
    cd ~/.vim # cd to repo
    git clone http://github.com/gmarik/vundle.git ~/.vim/vundle.git # get vundle
    vim vimrc # install the bundles
        :BundleInstall
    ln -s ~/.vim/vimrc ~/.vimrc # make vimrc known
